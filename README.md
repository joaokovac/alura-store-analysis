Challenge Alura Store - Análise de Dados

Sobre o projeto

Este projeto foi desenvolvido como parte do Challenge de Data Science da Alura.

O objetivo da análise é ajudar o Sr. João a decidir qual das quatro lojas da rede Alura Store deve ser vendida para que ele possa investir em um novo negócio. Para isso, foram analisados dados de vendas, avaliações de clientes, produtos e custos de frete de cada loja.

A partir dessas informações, foi possível comparar o desempenho das lojas e identificar qual apresenta menor performance em relação às demais.

Métricas analisadas

Durante a análise dos dados foram considerados os seguintes indicadores:
	•	Faturamento total de cada loja
	•	Categorias de produtos mais populares
	•	Média de avaliação dos clientes
	•	Produtos mais vendidos
	•	Produtos menos vendidos
	•	Custo médio de frete

Ferramentas utilizadas
	•	Python
	•	Google Colab
	•	Matplotlib

Conclusão

Com base nos dados analisados foi possível comparar o desempenho das quatro lojas e identificar qual apresenta menor desempenho geral, auxiliando na decisão sobre qual loja poderia ser vendida.
